# RetroMan Design System

## Overview
RetroMan is a physical media collection app with an 80s/90s retro synthwave aesthetic that supports both light and dark themes.

## Color Palette

### Dark Theme (Primary)
- Background: #0d0b1e (deep purple-black)
- Foreground: #e8e6f2 (soft white)
- Primary: #ff2d95 (neon pink/magenta)
- Accent: #00f5d4 (neon cyan/teal)
- Secondary: #2a2545 (muted purple)
- Muted: #232040 (dark purple)
- Border: #3d3860 (purple-gray)
- Neon Yellow: #ffd700 (for highlights)

### Light Theme
- Background: #faf8ff (soft lavender white)
- Foreground: #1a1625 (deep purple-black)
- Primary: #d4177a (rich magenta)
- Accent: #0d9488 (teal)
- Secondary: #f0ecf9 (soft purple)
- Muted: #f5f3fa (light purple)
- Border: #e5e0f0 (soft purple-gray)

## Typography
- Display/Headings: Press Start 2P (pixel font for retro feel)
- Body: Geist Sans (modern, clean)
- Monospace: Geist Mono

## Visual Effects (Dark Mode)
- Neon glow on primary elements (text-shadow with pink/cyan)
- Scanline overlay on images for CRT effect
- Grid background pattern (40px spacing)
- Neon border glow on interactive elements

## Media Types & Icons
- Games: Gamepad2 icon
- Vinyl Records: Disc3 icon
- CDs: Disc icon
- Cassettes: CassetteTape icon
- VHS: Tv icon

## Condition Color Coding
- Mint: Green (#22c55e)
- Excellent: Cyan/Teal (accent color)
- Good: Yellow (#ffd700)
- Fair: Orange (#f97316)
- Poor: Red (destructive)

## Component Patterns
- Cards with subtle border, rounded-lg corners
- Hover states increase border opacity and add shadow
- Active states use primary color background
- Badges use rounded-full with semi-transparent backgrounds
- Grid layouts: 2-6 columns responsive
- List layouts: fixed-width columns for alignment

## Layout Structure
- Sticky header with logo, search, theme toggle, actions
- Collapsible sidebar (hidden on mobile) with collections
- Main content area with grid/list toggle
- Modal overlays for detail views

## Spacing Scale
- xs: 4px (gap-1)
- sm: 8px (gap-2)
- md: 16px (gap-4)
- lg: 24px (gap-6)
- xl: 32px (gap-8)

## Animation
- Transitions: 150-200ms ease
- Hover scale on grid items: scale-105
- Smooth opacity transitions for overlays
